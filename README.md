#   GitHubUserViewer

## このアプリについて
* GitHubのユーザーを閲覧できるクライアントアプリです

---
## セットアップ
* Homebrewがない場合はインストール 
`https://brew.sh/index_ja`
* homebrewでrswiftをインストール
`$ brew install rswift`
* コミットテンプレートの設定
`$ sh ./Scripts/committemplete.sh`

---
## ライブラリ
* Swift Package Managerで管理しています
`https://swift.org/package-manager/`
